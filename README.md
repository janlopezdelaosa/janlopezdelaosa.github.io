# Weather Forecast

Live demo: https://janlopezdelaosa.github.io
[API reference](./back/API.md)

## Run locally

Clone repo and

```sh
$ npm install
```

> npm 7 is required to support workspaces

### Frontend

```sh
$ cd front
$ npm run start
```

### Backend

```sh
$ cd back
$ nodemon index.js
```

### Tests

```sh
$ cd front
$ npm run cypress:open
```

Click `main-cases_spec.js`.
